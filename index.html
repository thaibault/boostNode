boostNode

boostNode is a high level python library. This library supports python2.7+ and python3.3+ environements.

Features

  • platform independent webbased gui toolkit
  • smart, very secure multiprocessing webserver
  • macro processor for text based files
  • file synchronisation
  • many extended language feature like signature checking or jointpoints for aspect orientated programming
  • and much more..

Usage

Copy this folder to your projects directory and write something like:

import boostNode.extension.native
                # some stuff using imported boostNode components...

For advanced usage see the recommended module pattern described in "path/to/boosNode/init.py".

BoostNode is able to witch itself between python2.X and python3.X. To switch boostNode version between python3.X and python 2.X use this command:

/path/to/boostNode/runnable/macro.py -p /path/to/boostNode -e py
