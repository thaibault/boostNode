boostnode.paradigm.objectOrientation¶

This module implements features like getter and setter methods for implementing object orientated code.

class boostnode.paradigm.objectOrientation.Class¶

The main class which is intended for passing on other class. It serves a scope for one application to minimize conflicts with other classes in the global scope.

is_method(name)¶

Determines if the given class attribute is a callable method or something else.

name - function name to check

Returns “True” if the given attribute is a method or “False” otherwise.

Examples:

>>> Class().is_method('is_method')
True
>>> class A(Class): pass
>>> A().is_method('is_method')
True
>>> Class().is_method('not existing')
False
is_property(name)¶

Determines if the given class attribute is a property or something else.

name - property name to check

Returns “True” if the given attribute is a property or “False” otherwise.

Examples:

>>> Class().is_property('is_property')
False
>>> Class.test = 'A'
>>> Class().is_property('test')
True
>>> class A(Class): pass
>>> A().is_property('test')
True
classmethod pseudo_property(function)¶

Attaches a property to given function for indicating that given function handles read access to corresponding property.

function - Function which should act as property.

Returns given function to be usable as decorator.
