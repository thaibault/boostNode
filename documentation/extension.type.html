/data/scripts/python/maintained/boostNode/extension/type.py

This module provides mostly dynamic types/classes for checking function
call's against a given signature. Dynamic classes means they depend on
their context.

__getattr__(self, name)
Is triggered if a property was tried to be read but is
inaccessible.
 
"name" is the inaccessible property name.
 
Should return the current value of the given
property name depends on its getter method.
Returns "None" if no getter method is accessible.
 
Examples:
